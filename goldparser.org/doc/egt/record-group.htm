Group Record

Format

Group records occur after all the Symbol Records. The record is preceded by a byte field that contains the value 103, the ASCII code for the letter 'g'.

Fields

Initial Fields

Name Type Description
Index Integer The table index of the group in the Group Table. The values are zero-indexed.
Name String The name of the group.
Container Index Integer Index in the Symbol Table of the group's container symbol
Start Index Integer Index in the Symbol Table of the group's start symbol
End Index Integer Index in the Symbol Table of the group's end symbol
Advance Mode Integer Contains a constant indicating how the group will advance. See below.
Ending Mode Integer Contains a constant indicating how the group will handle the end symbol. See below.
Reserved Empty Reserved for future use.
Nesting 1 .. n   Described Below.

Nesting

The end of the record contains zero or more group indexes. These are allowed to be nested within the current group

Name Type Description
Index Integer The table index of the group in the Group Table. The values are zero-indexed.

Advance Mode

Index Property Name Description
0 Token The group will advance a token at a time.
1 Character The group will advance by just one character at a time.

Ending Mode

Index Property Name Description
0 Open The ending symbol will be left on the input queue.
1 Closed The ending symbol will be consumed.
