GOLD Parser ActiveX DLL Documentation

Version 2.5.0

 

Overview

This section contains documentation on the ActiveX DLL version of the GOLD Parser Engine.

The "GOLD Parser Engine.DLL" file contains the collection of objects required to organize and parse a source string or file. The parsing engine must first read the file created by the Builder application and once the tables are loaded the engine is ready perform.

The library contains the following objects:

  • GOLDParser Object
  • Rule Object
  • Reduction Object
  • Symbol Object
  • Token Object
