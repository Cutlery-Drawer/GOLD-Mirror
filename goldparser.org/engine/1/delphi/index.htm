The Delphi Programming Language evolved from the Pascal Programming Language with the guidance of the Borland Corporation. Control over Delphi later shifted to the Embarcadero Corporation.

Delphi Programming Language Homepage

Delphi contains a large number of class-library and language improvements over Pascal. The latest version of Delphi works with Microsoft's .NET Platform. As a result, modules developed in other .NET languages will work with Delphi.
