Vladimir Morozov is a software developer with more than 13 years of experience. His research interests are parser generators, programming languages and AI technologies. Vladimir lives and works in Kansas City, USA.

 He was amazed with the Devin's simple and clear approach to create parsers where grammar is written in a pure BNF notation and any programming language can be used to implement a parser. VBScript parser was created using GOLD Parser Builder and used to translate VBScript code files to VB.Net as a part of web based application conversion to .Net.

 He thanks Devin and other GOLD Parser project contributors for helping him with his project and he is glad to contribute VBScript grammar and C# parser engine which may be helpful for other developers.
